# MovieManagementSystem-StructVersion-Cpp
A  C++ project to store and manage movies with features like search, edit, delete, and save/load to file using dynamic arrays, file I/O, and structured data.
## Features
- Add new movies
- Search movies by title or director
- Edit movie information
- Delete movies
- List all movies
- Save data to file
- Load data from file
- Dynamic array resizing

## Movie Information Stored
- Movie Title
- Domestic Distributor
- Budget
- Domestic Box Office
- Domestic Release Date
- Director Name
- Running Time

## Technologies
- C++
- File Handling
- Dynamic Memory
- Structures
- Input Validation
